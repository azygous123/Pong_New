# Pong_New (Unity)

## Quick Launch

1. Download or extract the project files.
2. Open the project folder.
3. Navigate to the `Build` folder.
4. Double-click `Pong_New.exe` to run.

### Controls

**Player 1**
- W – Move paddle up
- S – Move paddle down

**Player 2**
- Up Arrow – Move paddle up
- Down Arrow – Move paddle down

**General**
- H – Toggle help screen (pauses and unpauses gameplay)
- ESC – Quit game to desktop



## Compiling / Running the Code in Unity

Updating the code or rebuilding the game requires Unity.

1. Download the project files (either as a ZIP file or by cloning the Git repository).
2. Extract the files into a folder on your computer.
3. Open **Unity Hub**.
4. Click **Open Project**.
5. Select the folder containing the project files.
6. Unity will open the project in the Unity Editor.
7. Press the **Play** button to run the game inside of Unity.
8. To create a new executable, select:
   - **File → Build and Run** or Ctrl + B
   - Follow the prompts to choose your build settings and location.

Unity will generate an executable file along with additional folders that must remain in the same directory for the game to run.


## Installing Unity

If Unity is not already installed (This game was built in Unity 6):

1. Go to https://unity.com/download
2. Download and install **Unity Hub**.
3. Open Unity Hub and install a Unity Editor version 6 is required for this project.
4. Once installation is complete open and run the project using the steps above.
